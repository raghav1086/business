## 3. Typography

### Primary Font Family

**English Text:**
- **Headings**: Inter, Poppins, or Montserrat (Modern, clean, professional)
- **Body**: Inter, Roboto, or Open Sans (Readable, friendly)

**Sanskrit/Marathi Text:**
- **Script**: Noto Sans Devanagari, Mukta, or similar (Clear, authentic)
- **Fallback**: System fonts with Devanagari support

### Typography Scale

**Headings:**
- H1: 48px / 3rem (Hero sections)
- H2: 36px / 2.25rem (Section headers)
- H3: 30px / 1.875rem (Subsections)
- H4: 24px / 1.5rem (Card titles)
- H5: 20px / 1.25rem (Small headers)
- H6: 18px / 1.125rem (Labels)

**Body:**
- Large: 18px / 1.125rem (Lead text)
- Regular: 16px / 1rem (Body text)
- Small: 14px / 0.875rem (Captions, metadata)
- Tiny: 12px / 0.75rem (Fine print)

### Typography Guidelines

**Do:**
- ‚úÖ Use consistent font sizes
- ‚úÖ Maintain proper line height (1.5-1.75 for body)
- ‚úÖ Ensure sufficient contrast (WCAG AA minimum)
- ‚úÖ Use font weights appropriately (400 regular, 600 semibold, 700 bold)

**Don't:**
- ‚ùå Mix too many font families
- ‚ùå Use fonts smaller than 12px
- ‚ùå Use all caps for body text
- ‚ùå Compromise readability for style

## 4. Imagery & Photography

### Style Guidelines

**Photography Style:**
- **Authentic**: Real Indian business owners, real scenarios
- **Optimistic**: Positive, growth-oriented, hopeful
- **Diverse**: Represent different regions, business types, ages
- **Professional yet approachable**: Not overly corporate

**Illustration Style:**
- **Modern**: Clean, flat design or subtle gradients
- **Cultural**: Respectful inclusion of Indian elements
- **Purposeful**: Every illustration tells a story
- **Consistent**: Unified style across all materials

### Image Subjects

**Business Scenarios:**
- Shop owners using mobile app
- Manufacturing units with inventory
- Service providers creating invoices
- Family businesses working together
- Business growth and success moments

**Avoid:**
- ‚ùå Stock photos that look fake
- ‚ùå Overly staged scenarios
- ‚ùå Stereotypical representations
- ‚ùå Low-quality or pixelated images

## 5. Iconography

### Icon Style

**Design Principles:**
- **Simple**: Clean, minimal, recognizable
- **Consistent**: Unified stroke width, corner radius
- **Meaningful**: Icons that clearly represent their function
- **Scalable**: Work at 16px to 64px sizes

### Icon Library

**Business Icons:**
- Invoice/Document
- Inventory/Stock
- Accounting/Ledger
- GST/Compliance
- Growth/Chart
- Trust/Shield
- Mobile/Phone
- Offline/Cloud

**Cultural Icons (Subtle):**
- Lotus (purity, growth)
- Diya (light, prosperity)
- Swastika (auspiciousness, well-being) - used respectfully

### Icon Usage

- Use icons to support text, not replace it
- Maintain consistent size within groups
- Use same style (filled vs. outlined) consistently
- Ensure icons are accessible (sufficient contrast)

## 7. Social Media Assets

### Profile Picture

**Specifications:**
- Size: 400x400px (square)
- Format: PNG with transparent background
- Content: Logo mark (icon only, no text)
- Clear at small sizes (works as app icon too)

### Cover Image

**Specifications:**
- Size: 1500x500px (Facebook/LinkedIn)
- Format: JPG or PNG
- Content: Brand message + visual elements
- Text: Keep minimal, readable at all sizes

### Post Templates

**Standard Post:**
- Size: 1080x1080px (square) or 1080x1350px (portrait)
- Format: PNG or JPG
- Brand colors and fonts
- Consistent layout across posts

**Story Templates:**
- Size: 1080x1920px (vertical)
- Format: PNG
- Brand colors with text overlays
- Mobile-optimized

### Content Themes

1. **Educational**: GST tips, business advice, accounting basics
2. **Inspirational**: Success stories, Samruddhi philosophy
3. **Product**: Feature highlights, how-to guides
4. **Community**: User stories, testimonials
5. **Cultural**: Indian business values, festivals, traditions

- Key features
